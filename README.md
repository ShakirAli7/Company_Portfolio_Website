# Company Portfolio Website

A modern, fully responsive company portfolio website built with HTML5 and CSS3.

## Features

- Fully responsive design for all devices
- Modern gradient design with smooth animations
- Sticky navigation bar
- Service showcase section
- Team member profiles
- Contact form
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
  - Flexbox
  - CSS Grid
  - Animations & Transitions
  - Media Queries

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/company-portfolio-website.git
   ```

2. Navigate to the project folder:
   ```bash
   cd company-portfolio-website
   ```

3. Open `index.html` in your browser

## Project Structure

```
company-portfolio-website/
├── index.html
└── README.md
```

## Customization

### Change Company Name
Find and replace "TechNova" in `index.html` with your company name.

### Change Colors
Modify the gradient colors in the CSS section:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Update Content
Edit the text in each section (About, Services, Team, Contact) directly in `index.html`.

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
